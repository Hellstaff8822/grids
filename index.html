Grid Layout

Grid Layout - це модуль CSS, який надає розширені можливості для створення сіток і розташування елементів на сторінці веб-сайту. Використовуючи Grid Layout, ви можете легко контролювати розміщення та розміри елементів в вашому дизайні, створюючи складні лейаути для вашого контенту. Ось більше інформації про Grid Layout: Grid Layout дозволяє створювати двовимірні сітки, де ви можете визначити кількість рядків та стовпців, а також розміщувати елементи в кожній комірці сітки.

Головні поняття Grid Layout:

  • Grid Container: Елемент, який містить всі елементи сітки. Встановлюється з використанням
    CSS властивості display: grid; або display: inline-grid;.
  • Grid Item: Дочірні елементи внутрішнього контейнера сітки, які розміщуються в комірках сітки.
  • Grid Line: Лінії, які визначають межі рядків і стовпців сітки.
  • Grid Track: Рядок або стовпець сітки між двома послідовними граничними лініями.
  • Grid Cell: Одна комірка сітки, обмежена чотирма граничними лініями.

Основні властивості Grid Layout:

  • grid-template-columns та grid-template-rows: Визначають розміри та кількість рядків і стовпців сітки.
  • grid-gap або grid-row-gap та grid-column-gap: Встановлюють відстані між рядками та стовпцями сітки.
  • grid-template-areas: Дозволяє задавати іменовані області для елементів сітки.
  • grid-column та grid-row: Визначають позицію елемента в сітці за номерами рядка і стовпця.

Grid Layout є потужним інструментом для створення складних макетів і розміщення контенту на веб-сайтах. Він дозволяє робити це з більшою гнучкістю та простотою порівняно зі старими методами розташування, такими як float і inline-block.
